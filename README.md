# AFIX Services Website

Official website for AFIX Services — licensed general and direct contractors in Singapore.

**Built Right. Built to Last.**

## Deployment

Deployed via [Vercel](https://vercel.com) from this GitHub repository.

## Local Development

Just open `index.html` in any browser — no build step needed.

## Customise

- Replace phone/email placeholders in `index.html`
- Swap Unsplash portfolio images with your own project photos
- Update stats (years, projects, satisfaction) to reflect real numbers
